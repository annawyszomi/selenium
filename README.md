# Selenium

The repository contains autometed test of two following test scenarios:
* Verify that the user can register an account with valid value to all required fields
* Verify that the form contains four following fields: Your name, Email, Password, Re-enter password and submit button.


# Pre requirements: 
* JDK 1.8+ 
* Maven
* Chrome 
* Selenium Client (Java) 
* Selenium WebDriver for Chrome  
* IntelliJ IDE

# Run the test
1. Install pre requrements
2. Clone repository to IntelliJ
3. Add to IntelliJ dependecies Selenium jar files
4. Ensure that path in class RegisterTest, method setUp() point to the local Chrome Driver file
4. Run the tests in class RegisterTest.
